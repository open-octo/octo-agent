🎯 关键解耦设计
  • Sender 抽象 — agent 不知道 HTTP/SSE/JSON，一切协议细节止步于 app/sender.go
  • ToolExecutor 接口 — 新增工具 = 往 allTools 加一行，不动 agent/provider
  • UIController / ViewSink — 同一套 Event 流被 TUI、Web、IM 三种后端解析渲染
  • agent 是叶子包 — 零上层依赖，依赖方向严格单向
