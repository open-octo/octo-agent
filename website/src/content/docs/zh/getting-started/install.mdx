<TabItem label="Linux">

安装脚本会自动识别你的架构、下载对应的 release、校验 SHA-256，并把 `octo` 装进 `PATH`：

```bash
curl -fsSL https://octo-agent.dev/install.sh | sh
```

然后启动本地服务并在浏览器里完成引导：

```bash
octo serve -d                  # 后台运行本地服务
xdg-open http://127.0.0.1:8088 # 打开控制台
```

`127.0.0.1` 是回环地址，不需要访问密钥——页面会直接进入首次运行引导（选 provider、填 key）。之后可以用
`octo serve --stop` 停止服务。更喜欢用终端？直接运行 `octo` 即可。

预编译包（linux / darwin / windows × amd64 + arm64）和 `checksums.txt` 都在
[最新 release](https://github.com/open-octo/octo-agent/releases/latest) 里，你也可以手动下载。

</TabItem>
<TabItem label="macOS">

**安装脚本（命令行）**——和 Linux 一样的一行命令：

```bash
curl -fsSL https://octo-agent.dev/install.sh | sh
octo serve -d                  # 后台运行本地服务
open http://127.0.0.1:8088     # 打开控制台
```

**双击安装器**——从[最新 release](https://github.com/open-octo/octo-agent/releases/latest) 下载
`octo-setup.pkg`（一个通用包同时覆盖 Apple Silicon 和 Intel），双击运行。

<Steps>
1. Installer.app 只提供"仅为我安装"——不需要管理员密码。
2. 它会把 octo 装进 `~/Library/Application Support/octo`，加进 `PATH`（追加到
   `~/.zprofile` / `~/.bash_profile` / `~/.profile`），并注册一个 LaunchAgent，
   每次登录时自动启动 `octo serve -d`。
3. 安装完成后会自动启动服务并打开 `http://127.0.0.1:8088`，带你完成首次运行引导。
4. 要在终端里用，打开一个**新**终端窗口运行 `octo`。
</Steps>

<Aside type="caution" title="Gatekeeper 会报警，这是预期的">
安装器还没有经过公证，Gatekeeper 会提示"来自身份不明的开发者"——右键（Control 点击）文件 → **打开**，
或者去**系统设置 → 隐私与安全性 → 仍要打开**。`.pkg` 没有 App Store 那种卸载器；运行
`~/Library/Application\ Support/octo/uninstall.sh` 可以清除它装的所有东西。
</Aside>

</TabItem>
<TabItem label="Windows">

从[最新 release](https://github.com/open-octo/octo-agent/releases/latest) 下载 `octo-setup.exe`，双击运行。

<Steps>
1. 按用户级别安装（不需要管理员权限弹窗），把 `octo` 加进 `PATH`，并添加开始菜单项。
2. 安装完成后会在后台启动本地服务（`octo serve -d`）并打开
   `http://127.0.0.1:8088`，带你完成首次运行引导。
3. 服务也会注册为每次登录自动启动（用户级 `Run` 项，无窗口），所以重启后控制台依然可用。
4. 要在终端里用，打开一个**新**终端窗口运行 `octo`。
</Steps>

安装器通过 [SignPath Foundation](https://signpath.org/) 的免费开源项目签名；在某个 release 完成签名之前，
Windows SmartScreen 可能会提示"Windows 已保护你的电脑"——点击**更多信息 → 仍要运行**即可。像卸载其他软件
一样从"添加或删除程序"里卸载。

</TabItem>
</Tabs>

## 升级

```bash
octo upgrade         # 原地安装最新 release，经过 SHA-256 校验
octo upgrade --check # 只比较版本号
```

Web UI 的版本徽标提供同样的升级入口。

## 用 Go 或从源码安装

```bash
go install github.com/open-octo/octo-agent/cmd/octo@latest
```

```bash
git clone https://github.com/open-octo/octo-agent.git
cd octo-agent
make build       # 产出 ./octo
```
